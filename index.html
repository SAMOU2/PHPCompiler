Le Mini-Compilateur PHP est un projet développé par Samou Megharba et Amine Merzouk, étudiants en génie logiciel à l'Université Abderrahmane Mira de Béjaïa. Ce compilateur a été conçu dans le cadre d'un projet académique visant à maîtriser les concepts fondamentaux de la compilation : analyse lexicale, analyse syntaxique et gestion de tokens.

L'application permet d'analyser du code PHP en temps réel, de détecter les erreurs lexicales et syntaxiques, d'afficher une table des tokens détaillée, et offre une interface moderne avec fond animé et paramètres interactifs. Elle intègre également des fonctionnalités avancées telles que la gestion de projets, les statistiques de code, la documentation intégrée via tooltips, et un mode plein écran.

Notre objectif est de fournir un outil pédagogique et pratique pour les développeurs PHP débutants et confirmés, qui souhaitent comprendre le fonctionnement interne d'un compilateur ou simplement valider la syntaxe de leurs scripts. Le projet est entièrement open source et disponible sur GitHub.

Ce site web de présentation offre une démo interactive du tokeniseur, ainsi que toutes les informations nécessaires pour télécharger et utiliser l'application JavaFX. N'hésitez pas à nous contacter pour toute question ou suggestion d'amélioration.
